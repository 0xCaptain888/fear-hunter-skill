01

什么是 Agent Trade Kit？

OKX Agent Trade Kit 是 OKX 官方推出的开源 MCP 工具包，它将 AI 助手（如 Claude、Cursor）与你的 OKX 交易账户直接连接。你只需要用自然语言描述需求，AI 就能自动执行交易。

82 个工具

现货、合约、期权、策略委托、账户管理、网格机器人

本地安全

API Key 永不离开本地，HMAC-SHA256 本地签名

MCP 标准协议

接入 Claude / Cursor / VS Code 等所有 MCP 客户端

完全开源（MIT 协议），代码审计透明： github.com/okx/agent-trade-kit

03

配置模拟盘 API Key

获取模拟盘 API Key

  1. 登录 okx.com → 右上角头像 → API 管理
  2. 点击「创建 V5 API Key」→ 选择 模拟交易
  3. 权限勾选：读取 + 交易（不要勾提币）
  4. 设置 Passphrase（自定义密码短语），记住它
  5. 保存 API Key、Secret Key、Passphrase 三个值

方式一：交互式向导（推荐）

Terminal
$ okx config init # 向导会引导你完成配置，运行一次即可

方式二：手动创建配置文件

~/.okx/config.toml
default_profile = "demo" [profiles.demo] api_key = "your-demo-api-key" secret_key = "your-demo-secret-key" passphrase = "your-demo-passphrase" demo = true # demo=true → 模拟盘，不涉及真实资金

安全提示：切勿将 API Key / Secret Key / Passphrase 粘贴到 AI 对话框中。Agent Trade Kit 的密钥只保存在本地配置文件中，AI 模型永远无法读取。

四层安全机制

Agent Trade Kit 从架构层面确保你的资金安全。

本地优先安全

API Key 保存在本地 ~/.okx/config.toml。HMAC-SHA256 签名在本地完成。AI 模型永远无法访问你的密钥。

权限感知注册

启动时自动读取 API Key 权限。若密钥无交易权限，下单工具根本不会注册到 AI 的工具箱中。

模拟盘模式

使用 --demo 标志，所有交易在模拟环境完成。零真实资金风险，适合测试和学习。

风险标签 + 开源审计

涉及资金操作的工具标记 [CAUTION]，AI 执行前会确认。MIT 协议完全开源，代码可审计。
